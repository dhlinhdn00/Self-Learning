# Smallest Number With All Set Bits

## INFO

**Index**: 3370

**Level**: Easy

**Link**: [Daily Question 2025-10-29](https://leetcode.com/problems/smallest-number-with-all-set-bits/?envType=daily-question&envId=2025-10-29)

---

## DESCRIPTION

You are given a positive number n .

Return the smallest number x greater than or equal to n , such that the binary representation of x contains only set bits

## EXAMPLE

### Example 1

    Input:
    n = 5
    Output:
    7
    Explanation:
    The binary representation of 7 is "111".

### Example 2

    Input:
    n = 10
    Output:
    15
    Explanation:
    The binary representation of 15 is "1111".

### Example 3

    Input:
    n = 3
    Output:
    3
    Explanation:
    The binary representation of 3 is "11".

---

## CONTRAINTS

- 1 <= n <= 1000
